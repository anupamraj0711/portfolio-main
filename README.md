## 🚀 Installation (Local Setup)

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
